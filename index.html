k8s-simple-logs Helm Repository

A simple Kubernetes log aggregation utility that exposes logs from all pods in the namespace via HTTP.

Installation

# Add the Helm repository
helm repo add k8s-simple-logs https://fredsmith.github.io/k8s-simple-logs

# Update your local Helm chart repository cache
helm repo update

# Install the chart
helm install my-release k8s-simple-logs/k8s-simple-logs

# Install with custom values
helm install my-release k8s-simple-logs/k8s-simple-logs \
  --set logkey=mysecret \
  --set service.type=NodePort \
  -n my-namespace --create-namespace
